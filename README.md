# Paradise Nursery Shopping Application

## Project Description
A React-based e-commerce application for browsing and purchasing houseplants.

## Live Demo
ðŸ”— [View Live Application](https://vinuradg.github.io/e-plantShopping/)

## Features
- Landing page with company information
- 18 unique houseplants
- Shopping cart functionality
- Add, remove, and update cart items
- Real-time total calculations
- Responsive design

## Technologies Used
- React
- Redux Toolkit
- Vite
- GitHub Pages

## Installation
```bash
npm install
npm run dev
```

## Deployment
```bash
npm run deploy
```
